 Fake News Detector

Description

Fake News Detector ek AI-based application hai jo users ko help karta hai identify karne me ki koi news real hai ya fake. Ye misinformation ko reduce karne aur users ko informed decisions lene me madad karta hai.

 Features

* News text input
* Fake/Real detection
* Probability result display
* Basic keyword analysis / API integration

 Technologies Used

* HTML
* CSS
* JavaScript
* AI APIs (optional)

 How It Works

User news text enter karta hai, system us text ko analyze karta hai keywords ya API ke through. Phir ye result show karta hai ki news kitni real ya fake ho sakti hai probability ke form me.

 Why This Project?

* Social impact project
* AI-based concept
* Portfolio ke liye impressive

 This project helps in spreading awareness against fake news and misinformation.
